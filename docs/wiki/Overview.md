# Overview

Project: groovy-syseng-lab — a hands-on learning repo for Groovy system engineering.

Goals
- Learn to write small Groovy utilities for system inspection and automation.
- Practice Git workflows: branching, PRs, reviews, and repository Wiki.
- Follow basic security hygiene and create a reproducible runbook.

Repository structure (key folders)
- `scripts/` — example Groovy scripts (e.g. `monitor.groovy`)
- `docs/lessons/` — lesson content and exercises
- `docs/wiki/` — this set of wiki pages (can be copied to GitHub Wiki)

Prerequisites
- Java JDK 11+
- Groovy (or SDKMAN to install)
- Git and a GitHub account
