# Groovy System Engineering — Starter Project

Purpose: a hands-on starter to learn Groovy for system engineering, Git, PRs, Wiki, and basic security practices.

Goals:
- Build a small Groovy system-inspection script.
- Learn Git branching, PRs, and repository Wiki.
- Practice secure-checklist and PR review flow.

Prerequisites:
- Java JDK (11+)
- Groovy (latest stable) or SDKMAN
- Git and a GitHub account (for PRs/Wiki)

Quickstart:
1. Open this folder in your editor.
2. Read the first lesson: docs/lessons/lesson-1-git-basics.md
3. Run the example: `groovy scripts/monitor.groovy`

Next steps: follow the lessons in `docs/lessons/` and ask me to walk through git commands interactively.
